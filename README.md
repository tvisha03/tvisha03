## Featured Projects

### [BloomthAI](https://github.com/tvisha03/mental-health-assistant) | AI-Powered Mental Health Assistant
**Tech Stack:** Python, FastAPI, PostgreSQL, LangChain, ChromaDB, RAG

Context-aware mental health support system using RAG pipeline with ChromaDB for semantic search. Built FastAPI backend with JWT authentication, PostgreSQL for user data management, and real-time sentiment analysis for mood tracking.

### [ContractIQ](https://github.com/tvisha03/ContractIQ) | Legal Document Analysis Platform
**Tech Stack:** Python, NLP, spaCy, React, Hybrid NER

Automated legal document analyzer with custom NER models for entity extraction and risk scoring. Features contract comparison engine for clause-level analysis and interactive React dashboard for visualization of legal vulnerabilities.

### [DefectGAN](https://github.com/tvisha03/synthetic_data_augmentation_using_GANs) | Industrial Anomaly Detection
**Tech Stack:** PyTorch, GANs, Computer Vision, Transfer Learning

GAN-based synthetic data augmentation for rare defect class generation in industrial quality control. Achieved 95%+ detection accuracy on MVTec dataset using transfer learning and optimized for real-time inference.

### [MBFT-Lite](https://github.com/tvisha03/MBFT-Lite-Project-V2-) | Federated Phishing Detection System
**Tech Stack:** TensorFlow, TinyBERT, XGBoost, Federated Learning

Privacy-preserving phishing detection using federated learning with TinyBERT + XGBoost ensemble. Enables distributed training across client devices without data centralization while maintaining real-time URL classification accuracy.
